# TrabalhoMatDiscreta
Trabalho desenvolvido para a disciplina de Matemática Discreta.
O projeto consiste em um jogo de quiz sobre funções (injetoras, sobrejetoras, bijetoras, compostas e inversas), integrado a uma narrativa fantasiosa.

No jogo, o jogador assume o papel de um cavaleiro responsável por salvar um mundo fictício ameaçado pelo caos matemático, respondendo corretamente a desafios relacionados ao conteúdo.

O principal objetivo do projeto é tornar o aprendizado de matemática mais lúdico, interativo e atrativo.
